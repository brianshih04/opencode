Run multiple tasks in parallel using a team of agents.

Two modes:

1. **Leader mode** (`mode: "leader"`) — Describe a high-level goal. A Team Lead agent will automatically break it down into 2-5 parallel subtasks and execute them. Example:
   ```json
   { "mode": "leader", "goal": "Analyze this codebase for security issues and performance bottlenecks", "max_tasks": 3 }
   ```

2. **Parallel mode** (`mode: "parallel"`) — Manually specify exact tasks to run in parallel. Example:
   ```json
   { "mode": "parallel", "tasks": [{ "description": "Find TODOs", "prompt": "Search for all TODO comments", "agent": "general" }, { "description": "Check types", "prompt": "Run type check and report errors", "agent": "general" }] }
   ```

Leader mode is recommended when you want the AI to figure out the best task decomposition.
Use parallel mode when you know exactly what tasks to run.
